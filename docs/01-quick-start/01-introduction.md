# 项目简介

## 什么是 Chan.py？

Chan.py 是一个基于 Python 实现的**缠论技术分析框架**，用于股票、期货、数字货币等金融市场的走势分析。

缠论（缠中说禅理论）是一种独特的技术分析方法，通过对K线进行分形、笔、线段、中枢等层层递进的分析，来识别市场走势的结构和买卖点。

---

## 核心功能

### 公开版功能（当前代码）

| 功能模块 | 说明 |
|----------|------|
| **K线合并** | 处理包含关系，生成合并K线 |
| **笔的计算** | 识别顶底分形，形成笔 |
| **线段计算** | 基于笔计算线段（支持多种算法） |
| **中枢识别** | 计算走势中枢（支持合并） |
| **买卖点识别** | 识别1、2、3类买卖点 |
| **多级别联立** | 支持日线+60分钟等多级别分析 |
| **可视化绘图** | 静态图/动画回放 |
| **技术指标** | MACD、均线、布林线、RSI、KDJ、Demark |
| **多数据源** | BaoStock、CSV、Futu、CCXT |

### 完整版额外功能

| 功能模块 | 说明 |
|----------|------|
| **自定义策略** | 动力学买卖点（cbsp）开发框架 |
| **机器学习** | 500+特征、模型训练、买卖点评分 |
| **AutoML** | 超参数自动搜索 |
| **交易引擎** | 对接Futu交易API |
| **数据库** | MySQL/SQLite后端存储 |

## 适用场景

1. **技术分析研究**: 学习和研究缠论
2. **量化选股**: 批量筛选符合条件的股票
3. **策略回测**: 验证交易策略的有效性
4. **实盘辅助**: 辅助判断买卖时机
5. **教学演示**: 缠论教学的可视化工具

---

## 设计理念

1. **模块化**: 各功能模块独立，便于扩展
2. **可配置**: 丰富的配置项，适应不同分析需求
3. **高性能**: 缓存机制，适合批量计算
4. **可扩展**: 支持自定义数据源、笔算法、线段算法
